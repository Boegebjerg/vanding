# Vanding
Guide til Henrik for vanding af planter.
## Indenfor
Indenfor skal der vandes én gang om ugen. Alle planterne indenfor, udover orkidéerne, skal samlet set have én kande vand, fordelt nogenlunde efter størrelse. Orkidéerne skal have en lille smugle vand hver anden uge kun.
