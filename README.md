# Udenfor
Udenfor er der 3 planter i krukker. De to af dem er tomatplanter, og den tredje er et lille træ. De vil gerne have én fuld kande hver. Tomaterne vil også gerne have lidt gødning, så der må man gerne lige komme lidt gødning i kanden. Det står på køkkenbordet, man kan bruge låget til at måle med. De skal cirka have ét låg hver.
